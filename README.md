# **Puissance 4**

Un petit projet permettant de jouer au Puissance 4 directement dans le navigateur à l’aide d’un serveur web codé en Go.

--------------------

## **Structure du projet**

```
power4-web/
├─ main.go
├─ game/
│  └─ game.go
├─ web/
│  └─ handlers.go
└─ templates/
   └─ index.html
```

## **Prérequis**
- Go 1.22 ou supérieur installé

## **Installation**

Dans le terminal :

```bash
git clone <URL_DU_REPO>
cd power4-web
```

## **Exécution**

Dans le terminal :

```bash
go run .
```

## **Utilisation**
Une fois le serveur lancé, ouvrir un navigateur et aller à :
```
http://localhost:8080
```
