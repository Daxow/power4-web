## **Structure du projet**

```
power4-web/
├─ main.go
├─ game/
│  └─ game.go
├─ web/
│  └─ handlers.go
└─ templates/
   └─ index.html
```

## **Prérequis**
- Go 1.22 ou supérieur installé

## **Installation**

Dans le terminal :

```bash
git clone https://github.com/Daxow/power4-web.git
cd power4-web
```

## **Exécution**

Dans le terminal :

```bash
go run .
```

## **Utilisation**
Une fois le serveur lancé, ouvrir un navigateur et aller à :
```
http://localhost:8080
```
